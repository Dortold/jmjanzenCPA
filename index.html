about this website

Hello, and welcome. I decided to create this (very) simple website, firstly to acclimatize myself with GitHub (version control, branching, pull requests, etc), and secondly to provide a central location with an overview of some of the things I've learned or participated in over the years.

Unfortunately, GitHub Pages does not support PHP, MySQL, or any of the dynamic functionality I really wanted to practice, but I've still enjoyed poking around this repository, even if the resulting website is a static one.
